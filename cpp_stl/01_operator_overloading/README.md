## ex23
- 생성자를 이용한 타입 변환
- B타입의 객체에 A타입의 객체를 대입하면 B타입의 생성자가 있는지 확인 있으면 호출
~~~c
A a;
B b;

b = a;	// B(A a) 생성자를 호출해서 새로운 B 객체 생성
~~~
